# Lesson 04: The Founder Starts Hiring Leadership

The studio is ready to do more, but the founder should not own every department forever.

This chapter will introduce:

- `SpawnAgent`
- parent-child runtime relationships
- delegation via signals
- a more realistic startup shape with leadership roles

This is the first lesson where the company splits into specialized roles, but it still stays narrow on purpose.

The likely structure is:

- `BackofficeAgent` for finance, approvals, and admin constraints
- `RecruiterAgent` for hiring pipeline work

The founder remains the top-level coordinator, but no longer acts as the direct owner of every function in the company. The CTO comes in later. This chapter is just about the CEO getting operational help without adding too much at once.
