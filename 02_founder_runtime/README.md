# Lesson 02: The Founder Goes Live

The founder can now organize ideas locally, but the studio still has no runtime and cannot react to external events.

This chapter will introduce:

- a Jido instance module
- `AgentServer`
- signals and signal routing
- the runtime version of the same founder workflow

Storywise, this is where the founder stops acting like a local notebook and starts acting like a live system that can react to studio events such as greenlighting an idea or planning a milestone.
